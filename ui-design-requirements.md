# 计算化学 AI Agent 平台 UI 设计需求文档

## 1. 设计概述

### 1.1 项目背景
计算化学和模拟计算领域需要一个专项AI Agent，解决实验化学家不熟悉代码规范和大模型使用的问题。通过把学术规范、计算库调用方式、参数设置规则沉淀为知识库，让做合成的同学也能自主完成模拟计算。

### 1.2 设计目标
- 创建**简洁、专业、高效**的计算化学AI Agent界面
- 确保**一致的用户体验** across all pages
- 支持**响应式设计**，适配桌面端和移动端
- 遵循**现代UI设计规范**，提升用户满意度
- **降低使用门槛**，让非编程背景的化学家也能使用

### 1.3 目标用户
- **实验化学家**：不熟悉代码规范，需要可视化操作界面
- **计算化学家**：需要高效的任务管理和参数配置
- **学生**：学习计算化学，需要指导和示例
- **研究人员**：需要快速完成模拟计算任务

### 1.4 设计原则
- **简洁性**：界面清晰，无冗余元素
- **一致性**：统一的视觉语言和交互模式
- **可访问性**：支持键盘导航和屏幕阅读器
- **响应性**：快速响应用户操作
- **专业性**：符合计算化学领域规范
- **易用性**：降低非专业用户的使用门槛

### 2.2 布局规范

## 3. 核心页面设计

### 3.1 仪表板页面
#### 3.1.1 概览卡片区域
- **布局**：4列网格布局
- **卡片内容**：
  - 活跃计算任务数
  - 今日完成任务数
  - 知识库条目数
  - 系统状态指示器
- **卡片样式**：
  - 背景色：`var(--color-background-secondary)`
  - 圆角：`var(--border-radius-md)`
  - 内边距：16px

#### 3.1.2 计算任务状态区域
- **布局**：3列网格布局
- **状态指示**：
  - 运行中：绿色圆点
  - 排队中：黄色圆点
  - 失败：红色圆点
  - 完成：蓝色圆点
- **信息显示**：
  - 任务名称
  - 计算类型
  - 进度百分比
  - 预计剩余时间

#### 3.1.3 最近任务区域
- **布局**：垂直列表
- **任务项信息**：
  - 任务名称
  - 计算类型
  - 状态标签
  - 创建时间
  - 操作按钮（查看、编辑、删除）

### 3.2 计算任务管理页面
#### 3.2.1 任务列表
- **布局**：表格视图
- **列信息**：
  - 任务ID
  - 任务名称
  - 计算类型
  - 状态
  - 创建时间
  - 完成时间
  - 操作按钮

#### 3.2.2 创建任务页面
- **表单布局**：垂直表单
- **配置项**：
  - **基本信息**：
    - 任务名称
    - 计算类型（分子动力学、DPD、量子化学等）
    - 描述信息
  - **分子结构**：
    - 结构文件上传（PDB、MOL、XYZ等）
    - 结构可视化预览
    - 结构编辑器
  - **计算参数**：
    - 力场选择（AMBER、CHARMM、OPLS等）
    - 温度、压力设置
    - 时间步长
    - 总步数
    - 输出频率
  - **高级参数**：
    - 边界条件
    - 静电作用处理
    - 温控/压控方法
    - 约束设置

#### 3.2.3 任务详情页面
- **布局**：详情页面
- **内容区域**：
  - 任务基本信息
  - 计算参数配置
  - 运行状态监控
  - 输出文件列表
  - 结果可视化

### 3.3 知识库管理页面
#### 3.3.1 知识库浏览
- **布局**：卡片网格或树形结构
- **分类**：
  - 学术规范
  - 计算库使用指南
  - 参数设置规则
  - 常见问题解答
  - 示例代码
- **搜索功能**：
  - 关键词搜索
  - 分类筛选
  - 标签筛选

#### 3.3.2 知识条目详情
- **布局**：详情页面
- **内容区域**：
  - 标题
  - 分类标签
  - 内容描述
  - 代码示例
  - 参数说明
  - 注意事项
  - 相关条目

#### 3.3.3 知识库管理
- **功能**：
  - 添加新条目
  - 编辑现有条目
  - 删除条目
  - 导入/导出
  - 版本管理

### 3.4 代码生成与验证页面
#### 3.4.1 代码生成界面
- **布局**：分栏布局
- **左侧**：参数配置面板
- **右侧**：生成的代码预览
- **功能**：
  - 选择计算类型
  - 配置参数
  - 生成代码
  - 代码高亮显示
  - 复制代码
  - 下载代码

#### 3.4.2 代码验证界面
- **布局**：分栏布局
- **左侧**：代码输入区域
- **右侧**：验证结果
- **验证内容**：
  - 语法检查
  - 规范检查
  - 库调用检查
  - 参数合理性检查
- **验证结果**：
  - 错误列表
  - 警告列表
  - 建议列表

### 3.5 结果可视化页面
#### 3.5.1 分子结构可视化
- **布局**：3D可视化区域
- **功能**：
  - 分子结构旋转、缩放
  - 原子标签显示
  - 键长、键角测量
  - 截面视图
  - 动画播放

#### 3.5.2 轨迹分析
- **布局**：图表区域
- **图表类型**：
  - 能量-时间曲线
  - 温度-时间曲线
  - 压力-时间曲线
  - RMSD-时间曲线
  - 径向分布函数
- **交互功能**：
  - 缩放、平移
  - 数据点查看
  - 图表导出

#### 3.5.3 统计分析
- **布局**：统计卡片
- **统计指标**：
  - 平均能量
  - 温度波动
  - 压力波动
  - 扩散系数
  - 粘度系数

### 3.6 对话界面
#### 3.6.1 消息区域
- **布局**：垂直滚动列表
- **消息类型**：
  - 用户消息：右对齐，蓝色背景
  - Agent消息：左对齐，灰色背景
  - 系统消息：居中，浅灰色背景
  - 代码块：特殊样式，带复制按钮
- **消息内容**：
  - 文本内容
  - 代码块
  - 图表
  - 分子结构
  - 时间戳

#### 3.6.2 输入区域
- **布局**：底部固定
- **输入组件**：
  - 文本输入框（支持多行）
  - 发送按钮
  - 附件按钮（支持分子结构文件）
  - 代码按钮
  - 知识库按钮
  - 模板按钮

## 4. 组件设计规范

### 4.1 按钮组件
#### 4.1.1 主要按钮
- **背景色**：`var(--color-background-info)`
- **文字色**：`var(--color-text-info)`
- **圆角**：`var(--border-radius-md)`
- **内边距**：10px 16px
- **悬停效果**：轻微透明度变化

#### 4.1.2 次要按钮
- **背景色**：透明
- **边框**：0.5px solid `var(--color-border-secondary)`
- **文字色**：`var(--color-text-primary)`
- **圆角**：`var(--border-radius-md)`

### 4.2 输入框组件
- **边框**：0.5px solid `var(--color-border-tertiary)`
- **圆角**：`var(--border-radius-md)`
- **内边距**：10px 12px
- **聚焦状态**：边框颜色变为 `var(--color-border-info)`

### 4.3 卡片组件
- **背景色**：`var(--color-background-primary)`
- **边框**：0.5px solid `var(--color-border-tertiary)`
- **圆角**：`var(--border-radius-lg)`
- **内边距**：16px
- **阴影**：无（扁平设计）

### 4.4 状态指示器
- **运行中**：绿色圆点（8px）
- **排队中**：黄色圆点（8px）
- **失败**：红色圆点（8px）
- **完成**：蓝色圆点（8px）

### 4.5 代码块组件
- **背景色**：`var(--color-background-secondary)`
- **边框**：0.5px solid `var(--color-border-tertiary)`
- **圆角**：`var(--border-radius-md)`
- **内边距**：12px
- **字体**：等宽字体
- **语法高亮**：支持Python语法高亮
- **复制按钮**：右上角，点击复制代码

### 4.6 分子结构可视化组件
- **容器**：固定高度，可调整大小
- **背景**：白色或浅灰色
- **交互**：鼠标拖拽旋转，滚轮缩放
- **工具栏**：旋转、缩放、重置、测量工具

## 5. 视觉设计规范

### 5.1 颜色系统
#### 5.1.1 主色调
- **主色**：蓝色系（用于主要操作和强调）
- **辅助色**：灰色系（用于背景和边框）
- **状态色**：绿色（成功）、黄色（警告）、红色（错误）

#### 5.1.2 颜色变量
```css
--color-background-primary: white
--color-background-secondary: #f8f9fa
--color-background-info: #e3f2fd
--color-text-primary: #212529
--color-text-secondary: #6c757d
--color-text-info: #1976d2
--color-border-tertiary: rgba(0,0,0,0.15)
--color-border-info: #1976d2
```

### 5.2 字体系统
#### 5.2.1 字体族
- **主字体**：`-apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif`
- **等宽字体**：`"SF Mono", Monaco, monospace`

#### 5.2.2 字体大小
- **标题**：15px（h1）、14px（h2）、13px（h3）
- **正文**：13px
- **小字**：12px
- **极小字**：11px

#### 5.2.3 字体粗细
- **常规**：400
- **中等**：500
- **粗体**：600（谨慎使用）

### 5.3 间距系统
#### 5.3.1 基础间距
- **4px**：极小间距
- **8px**：小间距
- **12px**：中间距
- **16px**：大间距
- **24px**：超大间距

#### 5.3.2 组件间距
- **卡片内边距**：16px
- **表单元素间距**：12px
- **按钮间距**：8px
- **列表项间距**：12px

### 5.4 圆角系统
- **小圆角**：4px（按钮、输入框）
- **中圆角**：8px（卡片、容器）
- **大圆角**：12px（弹窗、模态框）

## 6. 交互设计规范

### 6.1 状态反馈
#### 6.1.1 加载状态
- **骨架屏**：用于内容加载
- **旋转图标**：用于操作处理
- **进度条**：用于长时间计算任务

#### 6.1.2 成功状态
- **提示消息**：绿色背景，显示2-3秒
- **图标变化**：按钮图标变为对勾
- **页面跳转**：成功后跳转到相关页面

#### 6.1.3 错误状态
- **错误提示**：红色背景，显示错误信息
- **表单验证**：输入框边框变红，显示错误提示
- **重试机制**：提供重试按钮

### 6.2 动画效果
#### 6.2.1 过渡动画
- **时长**：200-300ms
- **缓动函数**：ease-in-out
- **属性**：opacity、transform

#### 6.2.2 交互反馈
- **悬停效果**：轻微放大或颜色变化
- **点击效果**：轻微缩小
- **焦点状态**：边框颜色变化

### 6.3 响应式设计
#### 6.3.1 断点设置
- **桌面端**：≥1024px
- **平板端**：768px-1023px
- **移动端**：<768px

#### 6.3.2 布局调整
- **桌面端**：多列布局
- **平板端**：两列布局
- **移动端**：单列布局，侧边栏折叠

## 7. 页面流程设计

### 7.1 用户登录流程
```
登录页面 → 输入凭据 → 验证 → 仪表板
```

### 7.2 计算任务创建流程
```
任务管理 → 创建任务 → 选择计算类型 → 上传分子结构 → 配置参数 → 生成代码 → 验证代码 → 提交任务 → 任务列表
```

### 7.3 知识库使用流程
```
知识库 → 浏览分类 → 搜索条目 → 查看详情 → 应用知识
```

### 7.4 代码生成流程
```
代码生成 → 选择计算类型 → 配置参数 → 生成代码 → 验证代码 → 复制/下载
```

### 7.5 结果分析流程
```
任务详情 → 查看输出 → 可视化分析 → 统计分析 → 导出结果
```
